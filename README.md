# Chatbot Pustaka Cendekia 🤖📚

Aplikasi Chatbot Pustaka Cendekia menggunakan Streamlit dan Google GenAI untuk asisten pintar perpustakaan sekolah. Proyek ini dirancang untuk memudahkan interaksi, pencarian informasi, serta layanan administrasi perpustakaan secara digital.

---

## 🚀 Fitur Utama
* **Asisten Pintar Berbasis AI:** Menjawab pertanyaan seputar operasional, aturan, dan informasi perpustakaan menggunakan Google GenAI.
* **Antarmuka Interaktif:** Menggunakan Streamlit yang ringan dan mudah diakses, bahkan melalui *smartphone*.
* **Integrasi Data:** Dirancang untuk mendukung ekosistem digitalisasi perpustakaan sekolah.
